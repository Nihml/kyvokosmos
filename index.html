Σχετικά με τον Server

Ο server μας είναι ένας Survival Multiplayer (SMP) server. Φιλοξενείται σε μια αξιόπιστη εταιρία με ελάχιστο ping.
  • Έχει επιπλέον features όπως νέα mobs και 15+ επιπλέον νέα biomes!
  • Ο server είναι διαθέσιμος από την έκδοση 1.20.2 μέχρι 1.20.6.
  • Έχει δικό του currency
  • Σύστημα register/login
  • Δυνατότητα παρακολούθησης μέσω Dynmap
  • Σύνδεση με Discord για προβολή του chat του server εδώ.
